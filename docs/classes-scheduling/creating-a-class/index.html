Creating a Class

Last updated April 2026 · 6 min read

What you'll need

You need Manager or Owner access to create classes. Instructors can view the schedule but cannot create or edit class templates. If you're an instructor who needs to create classes, ask your studio manager to update your role.

Before you begin: Make sure you have at least one instructor added to your studio and your timezone is correctly configured. Both affect how the class displays to clients.

Step 1 — Class type & name

Navigate to Classes → New Class.

  1. Select a Class Type. Available types include Group Class, Workshop, and Course. Group Class is for regular recurring sessions; Workshops are one-off events; Courses are multi-session series sold as a package.

  2. Enter a Class Name. This is the public-facing name that appears on your booking page and in client emails (e.g., "Wednesday Morning Flow" or "Reformer Foundations").

  3. Add a short Description (optional, up to 400 characters). This appears on the class detail view on your booking page. Include level, props needed, or what clients should expect.

Step 2 — Capacity & duration

  1. Set the Duration in minutes. Most classes run 45 or 60 minutes. This determines the end time shown to clients and controls scheduling conflicts for instructor availability.

  2. Set a Maximum Capacity — the total number of spots available per class. Once reached, new bookings go to the waitlist. See Capacity & Waitlists for waitlist configuration.

  3. Optionally set a Minimum Capacity. If the class does not reach this number a set number of hours before start, you can configure it to auto-cancel and notify enrolled clients.

Tip: Set your capacity slightly below your room's physical limit to leave a buffer for last-minute instructor or equipment changes.

Step 3 — Recurrence rules

For a Group Class, you can schedule it as a one-off or as a recurring series.

  1. Set the Start Date and Start Time for the first session.

  2. Toggle Recurring on if this class runs on a schedule. Select the frequency: weekly, biweekly, or custom days of the week.

  3. Set an End Date for the series, or leave it as "ongoing." Ongoing classes continue until manually ended or the instructor's contract ends.

  4. Review the Schedule Preview panel on the right — it shows the next 6 occurrences based on your settings before you save.

Step 4 — Instructor assignment

  1. Select the Primary Instructor from your staff list. Only staff with the Instructor or Manager role appear here.

  2. Optionally add a Backup Instructor. If you substitute the primary instructor, the backup's name is shown to clients on the updated confirmation.

  3. Enable Instructor Conflict Check to prevent double-booking the same instructor across overlapping classes or appointments.

Step 5 — Publish your class

  1. Review all settings in the Summary Panel on the right side of the screen.

  2. Toggle the class status to Published to make it visible on your public booking page. While in Draft, only staff can see it.

  3. Click Save & Publish. Enrolled clients will receive a booking confirmation email for any sessions they are auto-enrolled in (if applicable).

Heads up: Publishing a recurring class schedules all future occurrences immediately. Clients with active memberships or matching passes may be auto-enrolled depending on your enrollment settings.
