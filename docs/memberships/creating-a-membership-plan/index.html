Creating a Membership Plan

Last updated April 2026 · 6 min read

Before you begin

Memberships create recurring revenue by automatically billing clients on a regular cycle. Before creating your first plan, make sure your payment method is connected — see Connecting Your Payment Method.

Plan vs. Pass: Use a Membership for ongoing, recurring access. Use a Pass Pack for one-time purchases of a fixed credit block with an expiry date.

Step 1 — Plan name & description

Navigate to Memberships → Plans → New Plan.

  1. Enter a Plan Name that is clear and marketable (e.g., "Unlimited Monthly", "4-Class Weekly", "Founders Rate"). This is the name clients see when purchasing.

  2. Add a Plan Description — include what's covered, class types included, and any terms clients should know before subscribing.

  3. Set the Plan Color (optional). Color-coding plans makes it easy to identify members at a glance in the check-in flow and on the dashboard.

Step 2 — Billing frequency & amount

  1. Select the Billing Frequency: Weekly, Monthly, or Annual. Monthly is the most common for studio memberships.

  2. Set the Price for each billing cycle. If you offer an annual membership at a discount, enter the annual price and Obliq will calculate the monthly equivalent for display.

  3. Toggle Prorate First Payment if you want to charge clients only for the remaining days of the current billing period when they join mid-cycle.

  4. Set a Commitment Period if required (e.g., minimum 3-month commitment). Clients will be notified of the commitment before purchase.

Tip: For new studios, start with a simple Monthly Unlimited plan and a 4-Class Monthly plan. You can add more tiers once you understand what your clients want.

Step 3 — Class credits & access rules

  1. Set the Credits Per Cycle. For an unlimited plan, toggle Unlimited Credits. For limited plans, set the exact number of credits clients receive each billing period.

  2. Choose whether credits Roll Over to the next period or expire at cycle end. Rollover credits increase perceived value but can complicate financial projections.

  3. Set Class Type Restrictions if this membership only covers certain class types (e.g., group classes only, not workshops or private appointments).

Step 4 — Auto-renew settings

  1. Enable Auto-Renew to automatically charge the client's payment method at the start of each new billing cycle. This is on by default.

  2. Set the Renewal Reminder — send clients an email N days before their renewal date. 3–5 days is standard practice.

  3. Configure Failed Payment Handling: how many retry attempts to make, how many days between retries, and what happens to the member's access if payment ultimately fails (suspend or cancel).

Important: Once a client subscribes to a plan with auto-renew, they must be given the ability to cancel at least one full billing cycle before their next charge date. Make sure your cancellation window is visible and accessible in the client portal.

Step 5 — Pause & cancellation policy

  1. Enable Allow Member Pause if you want clients to be able to pause their membership. Set the maximum pause duration (e.g., up to 3 months per year).

  2. Set the Cancellation Notice Period — the number of days' notice required before cancellation takes effect. Common values are 7 or 14 days.

  3. Choose the Cancellation Effective Date: end of current billing period (most common) or immediately. Immediate cancellation is generally not recommended as it can result in partial billing disputes.

Publishing the plan

Toggle the plan to Active to make it available for purchase on your booking page and in the client portal. You can have multiple plans active simultaneously.
